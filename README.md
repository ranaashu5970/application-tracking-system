# 🚀 Application Tracking System

A full-stack MERN (MongoDB, Express.js, React.js, Node.js) Application Tracking System that helps candidates apply for jobs, recruiters manage applications, and coordinators streamline the hiring process.

---

## 📌 Features

- 🔐 User Authentication (Login/Register)
- 👨‍💼 Role-based Access (Candidate, Recruiter, Coordinator)
- 📄 Job Posting & Management
- 📊 Candidate Application Tracking
- 🧑‍💻 Recruiter Dashboard
- 📁 Manage Applications & Hiring Workflow

---

## 🛠️ Tech Stack

**Frontend:**
- React.js
- Tailwind CSS

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB
